# ТЕХБИЗ — Лендинг (HTML/CSS/JS)

Этот репозиторий содержит разработку лендинга компании ТЕХБИЗ.

## Цель проекта
Создать современный адаптивный лендинг, который:
- понятно объясняет услуги автоматизации гостиничного бизнеса;
- показывает кейсы и преимущества;
- приводит пользователя к заявке (форма/контакты/кнопка CTA).

## Технологии
- HTML5
- CSS3 (без фреймворков)
- JavaScript (vanilla)

## План работ
1. Документация и структура проекта.
2. Прототип блоков страницы (hero, услуги, кейсы, контакты).
3. Визуальный стиль и адаптивная верстка.
4. Интерактив на JavaScript (меню, анимации, валидация формы).
5. Финальная проверка, оптимизация и подготовка к публикации.

## Предполагаемая структура проекта
- index.html
- styles.css
- script.js
- docs/
  - landing-spec.md

## Контент, который нужно утвердить
- Краткое позиционирование компании ТЕХБИЗ (1-2 предложения): комплексная автоматизация отелей под ключ.
- Список услуг (утвержден):
  - автоматизация 1С для отеля;
  - подключение и настройка онлайн-касс;
  - настройка серверов и сетевой инфраструктуры;
  - интеграция замковых систем;
  - техническая поддержка и сопровождение.
- 2-4 кейса или примера выполненных работ.
- Контактные данные (телефон, email, мессенджеры, адрес).
- Логотип и фирменные цвета (если есть брендбук).

## Критерии готовности
- Корректно отображается на мобильных и десктопах.
- Понятная структура страницы и сильный призыв к действию.
- Форма контакта работает с базовой валидацией.
- Чистый и поддерживаемый код.

## SEO и индексация (реализовано)
- Семантическая структура страницы с корректной иерархией H1-H3.
- Уникальные `title`, `meta description`, `canonical`, Open Graph и Twitter Cards.
- JSON-LD микроразметка (`ProfessionalService`, `FAQPage`).
- Технические файлы индексации: `robots.txt` и `sitemap.xml`.
- Контент под поисковые запросы: автоматизация отелей 1С, кассы, серверы, замковые системы.

## Что сделать перед публикацией (обязательно)
1. Подставить реальный домен, телефон и email во все файлы.
2. Добавить реальную OG-картинку по адресу `https://ваш-домен/assets/og-cover.jpg`.
3. Подключить сайт к Яндекс Вебмастеру и Google Search Console.
4. Отправить sitemap на переобход и проверить статус индексации.
5. Установить HTTPS и включить редирект на основной канонический домен.

## Следующий шаг
После согласования документации переходим к созданию `index.html`, `styles.css` и `script.js`.
